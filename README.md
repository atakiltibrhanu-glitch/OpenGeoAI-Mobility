# OpenGeoAI-Mobility

OpenGeoAI-Mobility is an open GeoAI Urban Digital Twin framework for active mobility estimation under sparse sensing conditions.

## Features

- Spatial–Temporal Graph Transformer
- Sparse-sensor mobility estimation
- Unsensed location inference
- Streamlit Urban Digital Twin dashboard
- Future scenario simulation
- Pedestrian count estimation
- Sustainability-relevant error analysis
- SHAP explainability
